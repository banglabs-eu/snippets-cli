01 / Getting Started

One command to start taking notes

Install, connect, run.

1

Install

Run the install script. It clones the repo, sets up a virtual environment, installs dependencies, and configures everything interactively.

curl -fsSL https://raw.githubusercontent.com/banglabs-eu/cli.snippets/main/install.sh | bash

Requires Python 3.10+ and git. The installer will ask whether to connect to the API online or self-host with Docker.

Run the install script in PowerShell. It clones the repo, sets up a virtual environment, installs dependencies, and connects to the Snippets API.

irm https://raw.githubusercontent.com/banglabs-eu/cli.snippets/main/install.ps1 | iex

Requires Python 3.10+ and git.

02 / Features

Everything for structured note-taking

Capture, cite, tag, search, and export — all from one prompt.

⚡

Instant Capture

Just type text at the prompt — any unrecognized input becomes a note. No commands to memorize for basic note-taking.

📚

Source Citations

Link notes to bibliographic sources with MLA-style citations, authors, publishers, and edition tracking.

🏷

Smart Tagging

Tag notes with comma-separated labels. Autocomplete helps you reuse existing tags. Add or remove tags from any note.

🔖

Locator Tokens

Append p32 or t1:23:45 to notes to automatically parse page and time references into structured metadata.

📄

Markdown Export

Export notes by source, tag, or author as formatted Markdown files. View inline with bat or less.

⇥

Tab Completion

Context-aware autocomplete for commands, sources, tags, authors, and more. The REPL knows what you need.

🔍

Full-text Search

Find notes by content with find or f. Results render as Markdown and open in your pager.

✏

Inline Editing

Edit any note in place with e <id>. Arrow keys to navigate, Enter to save. Tracks last-edited timestamps.

🔒

Multi-user Auth

JWT login with invite-code registration. Per-user data isolation, password changes, and admin invite management.

✈

Offline Mode

Keep taking notes when the backend is unreachable. Notes are stored locally and automatically sync back when you reconnect.

🔐

Encrypted Local Cache

All local data — offline notes and autocomplete cache — is encrypted at rest with a password-derived key (PBKDF2 + Fernet).
