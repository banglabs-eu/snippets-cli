Snippets CLI Cheatsheet

Terminal note-taking with PostgreSQL — all commands at a glance

Notes

<text> Just type — any unrecognized input is saved as a noteThe cave allegory is profound p514
e / edit <id> Edit a note inline (arrow keys to modify, Enter to save)e 1
del <id> Delete a note by IDdel 5
find / f <query> Search notes by text (case-insensitive)find cave allegory

Sources

s <name_or_id> Set session source (Tab to autocomplete)s The Republic
s Show current source and citation
s clear Unset source — future notes have no source
ns <name> New source for session (reuse existing or create)ns The Republic
nse Source entry interview — MLA-ish fields with autocomplete
stadd <name> Add a new source typestadd Podcast

Tags

t <tags> Tag the last note (comma-separated)t philosophy, epistemology
s<id> +t <tags> Add tag(s) to a note by IDs2 +t cheese, bread
s<id> -t <tags> Remove tag(s) from a note by IDs2 -t cheese

Viewing / Export

b Browse all notes (rendered Markdown via bat/less)
vs <name_or_id> View/export notes by sourcevs The Republic
vt <tag> View/export notes by tagvt philosophy
va <Last, First> View/export notes by authorva Plato

System & Auth

login Log in to your account
register Create a new account (requires invite code)
logout Log out and clear saved token
whoami Show current logged-in username
passwd Change your password (also: change_password)
invite Generate an invite code (admin only)
invites List all invite codes (admin only)
help Show all available commands
exit Quit the REPL (also: quit)

Locator Tokens

Append to end of any note. Parsed automatically and stripped from body.

p32 Page 32
p.32 Page 32 (dot notation)
pp. 10-15 Pages 10–15 (range)
t1:23:45 Timestamp 1:23:45
t0:32 Timestamp 0:32

Tab Completion

<Tab> Command names (auth-aware)
s <Tab> Source names
t <Tab> Tag names (comma-separated)
s<id> +t <Tab> Tag names
vs <Tab> Source names
vt <Tab> Tag names
va <Tab> Author names
ns <Tab> Source names
nse fields Authors, publishers, cities, types

NSE Interview Fields

Prompted in order when running nse. Press Enter to skip any field.

Source titleRequired
Source typeBook, Article, Magazine, YouTube Video, Other
Year/datee.g. 2024, 380 BCE
URLWeb address
Accessed dateWhen you accessed the source
Editione.g. Revised, 3rd
Pages (range)e.g. 1-50
Extra notesFree text
Publisher namee.g. Penguin Classics
Publisher citye.g. London (if publisher given)
AuthorsLast + first name, repeat; empty to stop
